Portfolio

Below are selected projects that reflect my approach to UX design, branding, and front-end development.

Travel Destination Recommender

I worked on the Travel Destination Recommender as part of a group project that later transitioned into an individual assignment. After collaborating on the initial concept and structure, I cloned the repository and independently implemented the travel recommender API to enhance the application’s functionality. My contributions included designing and developing the Contact page, as well as helping shape the overall visual design and layout of the website. This project highlights my ability to collaborate within a team, take ownership of features, and apply UX principles to create a clear and user-friendly experience.

Movie Search App

The Movie Search App is a web application I created that allows users to search for movies and instantly view detailed information. The focus of this project was designing a simple, user-friendly interface that makes browsing and discovering movies feel intuitive. I paid close attention to layout, readability, and user flow while developing the front end with HTML, CSS, and JavaScript, making sure the experience stayed clear and responsive.
